      要求 #

      腕间图库支持实时预览手机中的图片，但必须满足下面要求。若无法满足要求或不想使用此功能，您可以在设置中关闭手机预览页面。

      • 手机为 Android 系统且必须安装腕间图库手机端。
      • 手表与手机已配对且通过蓝牙连接。

      安装手机端 #

      您可以通过下面任意链接下载手机安装包。

      错误排除 #

      • 确认手机已安装 相同版本 的腕间图库。
      • 确认手机与手表已经通过蓝牙连接。
      • 确认手机已经允许腕间图库自启或其他必要的权限。
